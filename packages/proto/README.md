# @sidcord/proto

Servisler arası gRPC iletişimi için protobuf tanımları.

## Yapı

```
proto/
├── user.proto      # Kullanıcı servisi
├── guild.proto     # Sunucu servisi
├── channel.proto   # Kanal servisi
├── message.proto   # Mesaj servisi
└── voice.proto     # Voice servisi
```

Faz 1'de doldurulacak.
